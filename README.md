## üì¶ Getting Started

To run the homepage locally and contribute, follow these steps:

1. **Clone the repository:**
   ```sh
   git clone https://github.com/The-A-P-O-L-L-O-Organization/Apollo-Org-Homepage.git
   ```
2. **Install dependencies** (requires [Node.js](https://nodejs.org/)):
   ```sh
   npm install
   ```
3. **Start the development server:**
   ```sh
   npm run dev
   ```
4. **Open** [http://localhost:3000](http://localhost:3000) in your browser.
